# 📦 StockFlow – Inventory Management System

A modern, multi-store Inventory Management System built using **HTML, CSS, and JavaScript**. StockFlow helps businesses efficiently manage inventory, sales, stores, staff, and reports through a clean and responsive dashboard.

---

## ✨ Features

### 🏪 Multi-Store Management

* Create and manage multiple stores
* Switch between stores
* Store-wise inventory tracking
* Store performance summaries

### 📦 Product Management

* Add, edit, and delete products
* Product image upload
* SKU/Product Code support
* Categories and suppliers
* Purchase and selling prices
* Low stock alerts
* Product status badges

### 📊 Dashboard & Analytics

* Total products overview
* Inventory value tracking
* Sales and profit insights
* Top-selling products
* Store-wise performance charts
* Monthly sales reports

### 💰 Sales Management

* Create sales records
* Automatic stock deduction
* Profit calculations
* Sales history

### 🧾 Invoice Generator

* Generate invoices after each sale
* Download invoices as PDF
* Customer and order details

### 👥 Staff Roles

Demo authentication system with role-based access:

#### Admin

* Full access to all features

#### Manager

* Manage stock and view reports

#### Staff

* View inventory and create sales

### 📋 Activity Logs

Track important actions such as:

* Product additions
* Stock updates
* Sales creation
* Store changes
* User activities

### 🔍 Search & Filters

* Search by product name or SKU
* Filter by category
* Filter by stock status
* Sort products easily

### 📄 Reports

* Sales reports
* Profit reports
* Inventory reports
* Low stock reports
* PDF export
* CSV export

### 📱 Responsive Design

* Mobile-friendly dashboard
* Hamburger navigation
* Bottom navigation on mobile
* Smooth animations
* Premium UI

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js
* html2pdf.js
* localStorage

---

## ⚠️ Note

This project uses **browser localStorage** for data persistence.

It is designed as a portfolio/demo project and does not require a backend or database.

Future versions can integrate Firebase or Supabase for real-time multi-user functionality.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/StockFlow-Inventory-Management-System.git
```

### Open Project

```bash
cd StockFlow-Inventory-Management-System
```

### Run

Simply open:

```text
index.html
```

No installation or backend setup required.
